pub fn resources_open_stream(
    path: &str,
    lookup_flags: ResourceLookupFlags,
) -> Result<InputStream, Error>
Expand description

Looks for a file at the specified @path in the set of globally registered resources and returns a InputStream that lets you read the data.

@lookup_flags controls the behaviour of the lookup.

§path

A path name inside the resource

§lookup_flags

A ResourceLookupFlags

§Returns

InputStream or NULL on error
