Struct DataOutputStream 

Source 
pub struct DataOutputStream { /* private fields */ }
Expand description

Data output stream implements OutputStream and includes functions for writing data directly to an output stream.

§Properties

§byte-order

Determines the byte ordering that is used when writing multi-byte entities (such as integers) to the stream.

Readable | Writeable

FilterOutputStream
§base-stream

The underlying base stream on which the I/O ops will be done.

Readable | Writeable | Construct Only

§close-base-stream

Whether the base stream should be closed when the filter stream is closed.

Readable | Writeable | Construct Only

§Implements

DataOutputStreamExt, FilterOutputStreamExt, OutputStreamExt, [trait@glib::ObjectExt], SeekableExt, OutputStreamExtManual

GLib type: GObject with reference counted clone semantics.
