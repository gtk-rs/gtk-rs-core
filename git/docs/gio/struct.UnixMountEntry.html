pub struct UnixMountEntry { /* private fields */ }
Expand description

Defines a Unix mount entry (e.g. /media/cdrom). This corresponds roughly to a mtab entry.

GLib type: Boxed type with copy-on-clone semantics.

pub fn for_mount_path( mount_path: impl AsRef<Path>, ) -> (Option<UnixMountEntry>, u64)

Gets a GioUnix::MountEntry for a given mount path.

If @time_read is set, it will be filled with a Unix timestamp for checking if the mounts have changed since with mount_entries_changed_since().

If more mounts have the same mount path, the last matching mount is returned.

This will return NULL if there is no mount point at @mount_path.
