Trait PropertyGet

pub trait PropertyGet {
    type Value;

    // Required method
    fn get<R, F: Fn(&Self::Value) -> R>(&self, f: F) -> R;
}
Expand description

A container type implementing this trait can be read by the default getter generated by the Properties macro.
