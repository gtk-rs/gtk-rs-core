A Dictionary entry.

While GVariant format allows a dictionary entry to be an independent type, typically youโll need to use this in a dictionary, which is simply an array of dictionary entries. The following code creates a dictionary:

use glib::variant::{Variant, FromVariant, DictEntry};

let entries = [
    DictEntry::new("uuid", 1000u32),
    DictEntry::new("guid", 1001u32),
];
let dict = entries.into_iter().collect::<Variant>();
assert_eq!(dict.n_children(), 2);
assert_eq!(dict.type_().as_str(), "a{su}");
