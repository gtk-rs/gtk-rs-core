Struct pango::Renderer

source · 
pub struct Renderer { /* private fields */ }
Expand description

Renderer is a base class for objects that can render text provided as GlyphString or Layout.

By subclassing Renderer and overriding operations such as @draw_glyphs and @draw_rectangle, renderers for particular font backends and destinations can be created.

This is an Abstract Base Class, you cannot instantiate it.
